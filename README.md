BookLovers

Для успішного запуску веб-додатку BookLovers необхідне встановлене середовище Node.js (рекомендована LTS версія) та пакетний менеджер npm або yarn.
Клонування репозиторію:
git clone https://github.com/ValeryKoshevaya/https---github.com-ValeryKoshevaya-booklovers.git
cd [Назва папки проєкту]
Запуск Backend (Серверна частина):
Сервер використовує Express.js і SQLite.
cd backend
npm install   # Встановлення залежностей
npm run start  Або "npm start". Запуск сервера на порту 3000 (за замовчуванням)
Запуск Frontend (Клієнтська частина):
Клієнтська частина розроблена на React з використанням Vite.
cd frontend
npm install   # Встановлення залежностей
npm run dev   # Запуск клієнта. Зазвичай доступний на http://localhost:5173
Проєкт буде доступний у браузері за вказаною адресою.
